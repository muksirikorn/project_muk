## Installing

Clone this project

```
git clone https://github.com/muksirikorn/project_muk.git
```

Navigate to project folder

```
cd project_muk
```

Create .env from .env-template and add all of it's values

```
nano .env
Key=Value
```

to run the project

```
flutter run
```
